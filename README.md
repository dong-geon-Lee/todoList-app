## :mag_right: &nbsp; Intro
> todoList 앱입니다. 기본적인 CRUD 기능을 제공합니다.</br>    
### 프로젝트의 실행 방법   
> $  npm install  
> $  npm start

### 유의사항   
> 깃페이지로 배포하여 새로고침과 일부 리다이렉트가 제한됩니다.  
> 인증과 CRUD는 잘 작동합니다.
<br/>
